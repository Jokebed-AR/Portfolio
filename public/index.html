Mi nombre es Jokebed Aguirre y estudio ingeniería cibernética en la Universidad Marista de Guadalajara. A continuación describiré brevemente algunos de los proyectos en los que he participado y algunos de ellos están en un repositorio github.

API Server

Asignatura:
Cloud Computing

Realicé una Api de prueba para consumir una Api de terceros. En esta utilicé la Api de Marvel (que cuenta con una documentación bastante robusta) para todos los métodos GET que puse. Para los métodos POST utilicé ApiToys (que tiene una documentación simple y corta)

Despertador Andante


Proyecto Integrador

Este consistía en un carrito fabricado con legos que funciona como una alarma. Cada vez que se activara la alarma emitía un sonido y el carrito comenzaba a avanzar hacia adelante, contaba con sensores de proximidad para que tuviera opción a moverse si se encontraba con un obstáculo. Este estaba programado en arduino y tenía una pantalla LCD donde se mostraba la hora.

Il Duomo Pizza

Asignatura:
Ingeniería de Software II

Este proyecto consistía en la realización de una página web de una pizzeria para que los clientes pudieran registrarse allí y realizar sus pedidos. Todos los datos de los clientes se quedaban registrados en una base de datos de SQL en la cual estuve trabajando.

Coffee Shop

Asignatura:
Programación Orientada a Objetos

Diseño de un CRUD en java para que un usuario pudiera realizar "pedidos" en una tienda de café.
